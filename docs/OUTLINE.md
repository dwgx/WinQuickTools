# 项目大纲

## 目标

做一套真正方便日常使用的 Windows 小工具，让重复、分散、需要点很多系统菜单的操作变成拖拽或双击即可完成。

## 用户场景

- 经常整理文件、下载目录、桌面文件的人
- 经常配置游戏、启动器、开发工具的人
- 不想每次都手动进属性、兼容性、注册表或控制面板的人
- 需要给朋友发一个简单工具解决问题的人

## 第一阶段

先把工具集合的基础框架搭起来，并完成 3 到 5 个低风险、高频使用的小工具。

优先级：

1. 批量设置以管理员身份运行
2. 批量复制路径工具
3. 批量文件名清理工具
4. 当前目录打开管理员终端
5. 网络连通性检查工具

## 第二阶段

完善工具体验。

- 每个工具加 `README.md`
- 每个工具加测试记录
- 统一中文提示格式
- 统一错误码
- 加一个总入口菜单
- 增加图标或快捷方式安装脚本

## 第三阶段

准备 GitHub 发布。

- 完成项目 README
- 补充截图或 GIF
- 选择 License
- 加 Release 打包说明
- 写清楚安全边界
- 建立 Issue 模板

## 不做什么

- 不做后台常驻程序
- 不做需要安装复杂依赖的软件
- 不做会静默删除大量文件的危险工具
- 不做绕过系统安全限制的工具
- 不收集用户数据
